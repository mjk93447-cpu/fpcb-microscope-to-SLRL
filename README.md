# fpcb-microscope-to-SLRL

Standalone Windows GUI for generating structured outputs (“structured latent representation candidates”) from microscopic FPCB crack images, plus human-in-the-loop contour labeling and training (to be implemented next).

## Subproject

- `fpcb_heatmap_app/` — current GUI app (baseline migrated from DREAM-SPTP)

## Quick start

```bash
cd fpcb_heatmap_app
pip install -r requirements.txt
python main.py
```
